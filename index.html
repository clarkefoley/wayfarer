Welcome to GitHub and the open source home of

"WayFarer: Never Settle" cloud-based application

The WayFarer: Never Settle application (Saas) was founded and developed by Team 3 in Professor Chengular-Smith's BITM 522 Spring 2012 evening MBA class. Sean Tooley, Rebecca Miller, Adam Silvia, and Clarke Foley co-founded the company to improve the process of booking travel.

Please download our source code as a .gz or .zip and fork it if you'd care to add a new feature or improve a current one. Bug fixes are always appreciated! Wayfarer Inc. is currently looking for talented developers. Submit a pull-request with your code snippets, and come work on our team! We accept virtual/remote workers, as well as co-located workers in our Troy, NY office.
